# 家庭影院 (Home Cinema)

自托管的家庭媒体服务器，用于管理和播放本地视频、照片等媒体文件。支持多用户权限控制，提供 Web 管理后台和 Android 客户端。

## 功能

### 媒体库

- 创建多个媒体库，支持**视频**和**相册**两种类型
- 指定本地目录路径，系统自动扫描索引媒体文件
- 支持目录层级浏览和关键词搜索
- 自动生成缩略图

### 视频播放

- 在线串流播放，支持拖拽进度条
- 自动记录播放进度，下次打开自动续播
- 手势控制：双击快进/快退、长按倍速、滑动调节音量和亮度
- 观看历史记录

### 文件上传

- 移动端支持选择文件上传到指定目录
- 上传队列管理，显示实时进度
- 上传完成后自动索引

### 用户与权限

- 管理员/普通用户两种角色
- 按媒体库分配访问权限，普通用户只能看到被授权的内容
- JWT Token 认证

### DLNA 投屏

- 自动发现局域网内的 DLNA 设备
- 投屏播放视频，支持远程控制

### 系统管理（Web 后台）

- 用户管理：增删、启用/禁用、密码重置、权限分配
- 任务队列：查看媒体库索引状态
- 操作日志：记录关键操作审计信息
- 系统仪表盘：存储、用户数等概览

## 快速开始

### 环境要求

- Go 1.25+
- Node.js
- Flutter 3.2+（构建移动端）
- ffmpeg / ffprobe（视频处理）

### 开发运行

```bash
make install      # 安装前端依赖（首次）
make dev          # 启动后端(:8080) + 前端(:3000)
```

### 构建

```bash
make build        # 构建单体二进制（前端嵌入Go二进制）
make release      # 跨平台编译 (linux/darwin/windows × amd64/arm64)
make release-apk  # 构建 Android APK
```

### 配置

| 环境变量 | 默认值 | 说明 |
|---|---|---|
| `JWT_SECRET` | `change-me-to-a-random-string` | JWT 签名密钥（生产环境务必修改） |
| `TOKEN_EXPIRE_HOURS` | `72` | Token 有效期（小时） |
| `DATA_DIR` | `./data` | 数据库存放目录 |
| `THUMBNAILS_DIR` | `./thumbnails` | 缩略图缓存目录 |
| `LISTEN_ADDR` | `:8080` | 监听地址 |

### 使用流程

1. 启动服务，首次访问进入初始化页面，创建管理员账号
2. 登录后台，创建媒体库并指定本地目录路径
3. 系统自动索引目录中的视频和图片
4. 创建普通用户并分配可访问的媒体库
5. Android 客户端配置服务器地址，登录即可浏览和播放
